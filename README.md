## рҹҢҢ ВҝQuГ© es Aurastral?

**Aurastral** es una herramienta de lГӯnea de comandos escrita en Ruby que genera horГіscopos personalizados usando inteligencia artificial (LLaMA 3.3 70B vГӯa Groq API gratuita). Con una experiencia de usuario completamente interactiva вҖ” splash screen animado, navegaciГіn con flechas y animaciones de carga вҖ” es el tipo de proyecto que nadie *necesita*, pero todos *quieren tener* en su GitHub.

---

## вңЁ ВҝQuГ© hay de nuevo en v0.0.2?

| # | Feature | DescripciГіn |
|---|---|---|
| рҹ”ҙ | **Splash screen animado** | ASCII art en rojo que aparece lГӯnea por lГӯnea al ejecutar `aurastral start` |
| рҹҺӣпёҸ | **MenГә interactivo completo** | Navega todas las opciones con flechas вҶ‘вҶ“ y Enter вҖ” sin escribir flags |
| вҸі | **Spinner de carga** | AnimaciГіn dorada mientras la IA consulta los astros |
| рҹҢҹ | **Nuevo comando de entrada** | `aurastral start` es el Гәnico punto de entrada |

## рҹҡҖ InstalaciГіn

### Prerequisitos

- Ruby `>= 3.2.0`
- Una API key gratuita de [Groq Console](https://console.groq.com/) вҖ” tarda 30 segundos

### Pasos

```bash
# 1. Clona el repo
git clone https://github.com/tu-usuario/aurastral.git
cd aurastral

# 2. Instala dependencias
bundle install

# 3. Configura tu API key
cp .env.example .env
# Edita .env y pega tu GROQ_API_KEY

# 4. Hazlo ejecutable
chmod +x bin/aurastral

# 5. ВЎLanza Aurastral!
bin/aurastral start
```

### InstalaciГіn global (opcional)

```bash
gem build aurastral.gemspec
gem install aurastral-0.0.2.gem

# Desde cualquier lugar:
aurastral start
```
